## Manzanita Research

A small lab in California. We make tools for people who play in bands and write DSP code and argue about gain staging — and increasingly, tools for anyone who cares about owning their own process.

Everything runs local. Everything ships open. We have zero interest in your data.

If it replaces the person, we're not interested. We build instruments.

### Music tools

| | Project | What it does |
|---|---|---|
| 🎵 | [audiotree](https://github.com/Manzanita-Research/audiotree) | React primitives for building audio apps — your component tree becomes your signal flow |
| 📖 | [glorybook](https://github.com/Manzanita-Research/glorybook) | Real-time setlist sync for bands |
| 🌳 | [grove](https://github.com/Manzanita-Research/grove) | 3D spatial audio mixer visualization — see sound in space |
| 💖 | [pedalboard](https://github.com/Manzanita-Research/pedalboard) | Open-source Claude Code skills for independent musicians |
| 🎛️ | [toneword](https://github.com/Manzanita-Research/toneword) | Semantic EQ — warmth, bite, air instead of frequency curves |
| 📻 | [village-radio](https://github.com/Manzanita-Research/village-radio) | Release Radar, but for your people |
